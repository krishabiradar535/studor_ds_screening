# Studor DS Screening Project

Project builds an end-to-end system to:
- Track student engagement
- Predict disengagement risk early
- Recommend suitable courses

## Setup Instructions

This project uses the Open University Learning Analytics Dataset (OULAD) 
downloaded automatically via the Kaggle API.

1. Go to kaggle.com → Profile → Settings → API Tokens → Generate New Token
2. Copy the token
3. Paste it in the input box when prompted (prompt comes after running 2nd cell of notebook)
4. The dataset will download and extract automatically

This approach was used to avoid committing large CSV files to the repository 
and ensure full reproducibility from a clean environment.
